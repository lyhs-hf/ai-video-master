# 工业级影视编剧核心法则·精简版

> 整合自：传统影视工业编剧理论精要
> 用于AI短视频创作场景的精简提炼

---

## 核心原则

**电影是现实生活的横截面**

- 故事从中间开始（不是从头讲起）
- 角色带着前史走进第一场戏
- 开场即态度（用什么方式看世界）

---

## 戏剧动作

**戏剧动作 = 目标(Goal) + 阻碍(Conflict)**

```
不是"一个人想喝水"
而是"沙漠中脱水的人找水"
```

检验标准：
- [ ] 目标是否合理且有急切性？
- [ ] 目标和阻碍之间是否有强烈冲突？
- [ ] 这个冲突是否足以驱动故事？

---

## 人物设计

### Want vs Need

| 概念 | 定义 |
|------|------|
| **Want（外在需求）** | 角色主动追求的目标 |
| **Need（内在欲望）** | 角色未意识到的灵魂缺口 |

Want和Need之间的张力是角色弧光的发动机。

### 人物弧光

**弧光 = 从A状态到B状态的不可逆变化**

### 矛盾性

好的角色至少有矛盾：外在表现 vs 内在真实

```
例：外在自信 vs 内在自卑
    外在冷漠 vs 内在渴望连接
```

---

## 视听化写作

### Show, Don't Tell

**严禁心理描写，只写能被看见的动作和能被听见的声音**

```
❌ 错误：他终于理解了对方的苦心
✅ 正确：他站了一会儿，把外套脱下来盖在对方身上
```

### 潜台词 (Subtext)

对话是冰山——说出来的话只是表面，真正的含义在水面之下

```
❌ 直白："我很害怕失去你"
✅ 潜台词："你走的时候把那件外套带上。……外面冷"
```

### 台词风格

- 口语化，像真人在说话
- 不同角色有不同说话方式
- 避免书面化的AI腔
- 不要让角色说出双方都知道的事

## 自检清单

### 核心概念与戏剧动作
- [ ] 是否建立了"目标+冲突"？
- [ ] 目标是否合理且急切？
- [ ] 冲突是否驱动故事？

### 人物塑造与弧光
- [ ] 外在需求和内在欲望是否清晰且对立？
- [ ] 是否预设了不可逆的内在变化？
- [ ] 结局是否展示了"新常态"？

### 场景写作与视听语言
- [ ] 是否有心理描写？（如有则Fail）
- [ ] 是否有括号暗示？（如有则Fail）
- [ ] 对话是否过于直白？
- [ ] 是否利用了潜台词和肢体语言？
- [ ] 台词是否口语化？
