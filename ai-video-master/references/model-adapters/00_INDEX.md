## 🎯 快速选择指南

### 图片模型选择

| 需求 | 推荐模型 | 理由 |
|------|---------|------|
| 电影感、质感人像 | Midjourney | 摄影术语支持强，材质表现好 |
| 创意场景、概念图 | DALL-E 3 | 自然语言理解强，画面感好 |
| 中文内容创作 | 即梦 | 中文理解最强，本土化 |
| 快速原型、电商图 | 通义万相 | 中文支持好，速度快 |

### 视频模型选择

| 需求 | 推荐模型 | 理由 |
|------|---------|------|
| 人物动作连贯 | 可灵 Kling | 人物动作自然，中文友好 |
| 情感表达、细腻动作 | 海螺 Hailuo | 情感表达好，动作自然 |
| 电影感运镜 | Runway | 运镜参数丰富，欧美风格 |
| 快速简单场景 | Pika | 生成速度快，上手简单 |

### 对口型模型选择

| 需求 | 推荐模型 | 理由 |
|------|---------|------|
| 专业级数字人 | HeyGen | 口型最准，质量最高 |
| 手机便捷操作 | 剪映 | 操作简单，随时可用 |
| 中文数字人 | 腾讯智影 | 中文支持好，数字人多 |

## 🔄 模型组合推荐

### 短视频制作标配流

```
Midjourney（封面/场景图）
    ↓
可灵 Kling（图生视频）
    ↓
HeyGen（对口型配音）
```

### 快速生产流

```
即梦（图片生成）
    ↓
海螺 Hailuo（图生视频）
    ↓
剪映（对口型剪辑）
```

### 电影质感流

```
Midjourney（电影感封面）
    ↓
Runway（电影运镜）
    ↓
HeyGen（专业配音）
```

## 📌 使用注意事项

1. **提示词语言**：中文内容优先使用国产模型（即梦、通义万相、可灵、海螺）
2. **图片比例**：短视频统一使用 **9:16（竖版）**
3. **风格统一**：同一视频系列保持同一模型，避免风格割裂
4. **高清素材**：HeyGen 等对口型工具对图片质量要求高，建议使用 1024px+ 图片
