# 图片模型适配指南

> 适用场景：分镜头封面图、场景图、人物图、道具图

---

## 1. Midjourney

### 🎯 定位
国际领先的 AI 生成图片模型，擅长电影感、质感人像、摄影级画面。

### 📝 提示词模板

**竖版短视频封面（9:16）**
```
[主题描述], cinematic lighting, shot on [焦段], f/[光圈], shallow depth of field, film grain, 9:16 vertical format, --ar 9:16 --s 250 --v 6.1 --style raw
```

**人物特写**
```
[人物描述], portrait photography, [情绪词汇], soft natural light, 85mm lens, bokeh background, 9:16 --ar 9:16 --s 300 --style raw
```

**场景图**
```
[场景描述], wide shot, golden hour lighting, cinematic color grading, ultra-detailed, 9:16 vertical --ar 9:16 --s 200 --style raw
```

### ⚙️ 参数设置建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| `--ar` | `9:16` | 竖版短视频必备 |
| `--s` | `200-350` | 风格化程度，越高越有艺术感 |
| `--v` | `6.1` | 使用最新模型版本 |
| `--style raw` | 必须加 | 减少过度处理，画面更自然 |
| `--no` | 按需添加 | 排除不需要的元素 |

**英文提示词格式**：
- 使用摄影术语（cinematic, bokeh, golden hour）
- 使用材质词汇（metallic, matte, glass, silk）
- 精确焦段（35mm, 85mm, 24mm）

**中文提示词格式**：
- 先写英文核心描述，再加中文补充
- 示例：`A young woman, portrait, golden hour, 85mm lens --ar 9:16 --s 300, 亚洲面孔，精致五官`

### ❌ 常见失败模式 & 解决方案

| 失败现象 | 原因 | 解决方案 |
|----------|------|----------|
| 人物脸模糊/扭曲 | `--s` 值过高或模型处理人像能力不足 | 降低 `--s` 至 150-200，或用 `--iw` 提高参考图权重 |
| 画面过暗/过曝 | 光线描述不准确 | 添加具体光线描述：`soft diffused light`, `golden hour backlighting` |
| 风格太AI感 | `--style raw` 未添加 | 必须加 `--style raw` 减少AI味 |
| 文字乱码 | 在提示词中加入文字 | 避免在 Midjourney 中生成文字，后期用剪映/PS添加 |
| 9:16比例构图奇怪 | 模型不擅长竖版构图 | 用 `--ar 9:16` 并在描述中强调 `vertical composition` |

### ✅ 优缺点分析

**优点**：
- 电影感、质感最强
- 摄影术语支持好，参数丰富
- 人物真实感强
- 风格多样可调

**缺点**：
- 需要英文提示词（中文理解有限）
- 速度较慢（需排队）
- 价格较高
- 有时会过度美化

## 2. DALL-E 3

### 🎯 定位
OpenAI 出品，自然语言理解最强，擅长创意场景、概念图、图文匹配。

### 📝 提示词模板

**竖版短视频封面（9:16）**
```
[详细场景描述], 9:16 vertical format, cinematic composition, 高质量摄影风格
```

**创意概念图**
```
[创意描述], 包含以下元素：[元素1]、[元素2], 风格：[风格描述], 9:16比例
```

**图文匹配封面**
```
[主题]主题配图，文字内容：[文案]，设计风格：[风格]，竖版 9:16
```

### ⚙️ 参数设置建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 比例 | `1024x1792` 或 `1024x1024` | 直接指定像素尺寸 |
| 质量 | `hd` | 高清模式 |
| 风格 | `natural` 或 `vivid` | natural 更真实，vivid 更生动 |

**提示词格式**：
- **自然语言优先**：用完整句子描述，不要堆砌关键词
- **画面感强**：描述场景氛围、光线、人物状态
- **中文友好**：可直接使用中文提示词

**示例**：
```
一个年轻女性坐在咖啡馆窗边，阳光透过玻璃洒在她脸上，她正在看手机，嘴角带着微笑，氛围温馨惬意，竖版构图，电影感
```

### ❌ 常见失败模式 & 解决方案

| 失败现象 | 原因 | 解决方案 |
|----------|------|----------|
| 构图不符合预期 | 提示词不够具体 | 补充具体构图描述：`前景/中景/后景`、`俯视/仰视` |
| 风格不统一 | 多次生成风格差异大 | 固定关键描述词，建立自己的风格词库 |
| 文字生成不稳定 | DALL-E 3 文字能力有限 | 不要依赖生成文字，后期叠加 |
| 细节不够 | 提示词太笼统 | 添加材质、光线、表情等细节描述 |

### ✅ 优缺点分析

**优点**：
- 自然语言理解最强
- 中文支持好
- 画面逻辑性强
- 无版权争议

**缺点**：
- 艺术感/电影感不如 Midjourney
- 参数调节空间小
- 生成速度慢
- 每次只能生成一张

## 3. 即梦（JiMeng）

### 🎯 定位
字节跳动出品，**中文理解最强**，本土化内容创作首选。

### 📝 提示词模板

**竖版短视频封面（9:16）**
```
中文描述：[主体描述]，电影感，9:16竖版，光线：[光线描述]，氛围：[氛围描述]
```

**人物图**
```
中文描述：[人物描述]，精致五官，真实质感，柔和光线，竖版构图，9:16
```

**场景图**
```
中文描述：[场景描述]，电影级画质，柔和打光，竖版 9:16，高清细节
```

### ⚙️ 参数设置建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 比例 | `9:16` | 直接选择竖版 |
| 画质 | `高清` | 必须选高清 |
| 模型版本 | 最新版 | 使用最新模型 |
| 参考图 | 可叠加 | 支持图生图 |

**中文提示词格式**：
- 直接使用中文，描述详细
- 加入情绪词：`温馨`、`压抑`、`活力`
- 加入风格词：`电影感`、`海报风`、`水彩风`
- 加入光线词：`逆光`、`柔光`、`霓虹`

**混合语言提示词**：
```
现代都市风格，一位穿西装的男士走在雨中，赛博朋克氛围，精致细节，9:16竖版
```

### ❌ 常见失败模式 & 解决方案

| 失败现象 | 原因 | 解决方案 |
|----------|------|----------|
| 人物网红脸 | 训练数据偏向 | 添加描述：`普通长相`、`真实质感`、`素人面孔` |
| 中文文字乱码 | 文字生成能力有限 | 后期叠加文字，不依赖模型生成 |
| 风格过于同质 | 模板化严重 | 混合不同风格词：`油画质感+摄影真实感` |
| 手部畸形 | 手指生成是难题 | 避免手部特写，或用局部重绘修复 |

### ✅ 优缺点分析

**优点**：
- **中文理解最强**，提示词直接有效
- 本土化内容表现好
- 价格便宜/免费额度多
- 支持中文文字生成（有限）
- 图生图功能强

**缺点**：
- 国际风格/电影感不如 MJ
- 人物真实性有时不稳定
- 英文提示词理解一般
- 部分功能需付费

## 4. 通义万相（Tongyi Wanxiang）

### 🎯 定位
阿里云通义系列，**中文支持好**，适合快速原型、电商图、内容批量生产。

### 📝 提示词模板

**竖版短视频封面（9:16）**
```
中文：[主体描述]，电影感，9:16竖版，光线柔和，细节丰富
```

**电商/产品图**
```
中文：[产品描述]，白底/场景图，高清细节，专业打光，竖版构图
```

**人物图**
```
中文：[人物描述]，真实质感，柔和自然光，精致五官，9:16比例
```

### ⚙️ 参数设置建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 比例 | `9:16` | 选择竖版 |
| 风格 | `默认`/`电影`/`电商` | 根据需求选择 |
| 质量 | `高清` | 优先高清 |

**中文提示词格式**：
- 简洁明确，不用复杂句式
- 关键词堆叠效果比完整句子好
- 示例：`美女，咖啡馆，阳光，窗边，微笑，竖版，电影感`

### ❌ 常见失败模式 & 解决方案

| 失败现象 | 原因 | 解决方案 |
|----------|------|----------|
| 画面偏灰/饱和度低 | 风格选择问题 | 切换到「生动」或「电影」风格 |
| 构图单一 | 模型偏好固定角度 | 多次生成选择，或用参考图引导 |
| 中文文字错误 | 文字生成不稳定 | 后期添加文字，不依赖模型 |
| 批量化后风格不一致 | 随机性大 | 建立自己的提示词模板包，固定核心词 |

### ✅ 优缺点分析

**优点**：
- 中文支持好
- 免费额度多
- 速度快
- 适合批量化生产
- 阿里云生态集成好

**缺点**：
- 艺术性/创意性一般
- 电影感不足
- 人物真实性有时欠佳
- 风格库较有限
