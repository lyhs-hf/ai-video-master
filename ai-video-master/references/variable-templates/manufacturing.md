# 工业制造模板

> 版本：1.0 | 日期：2026-04-07

---

## 模板A：成本型（节本降耗）

### 变量池

| 变量 | 内容 |
|------|------|
| **目标受众** | 工厂老板、生产总监（制造业企业） |
| **核心痛点** | ① 人工成本太高，利润率被压缩 ② 废品率高，浪费严重 |
| **破局道具** | ① 机器运转特写（高速但稳定） ② 电费/人工账单对比图 |

### 剧本设计

| 阶段 | 内容 |
|------|------|
| **黄金开场** | 反差特写：账单上红色的"人工成本"数字，被狠狠圈出来 |
| **核心道具** | 账单对比图（人工 vs 机器） |
| **情绪弧线** | 焦虑（成本太高）→ 绝望（利润被压薄）→ 转机（机器替代方案）→ 满意（成本下降） |

### 品牌金句

> "机器干重复活，人做有价值的事。"

### B-Roll建议

| 阶段 | 建议B-Roll |
|------|-----------|
| 压抑期 | 账单特写、高亮数字、工厂流水线工人 |
| 转机期 | 机械臂运转、机器特写、成品完美掉落 |
| 释放期 | 成本对比图表、工厂老板看数据满意点头 |

## 模板B：效率型（交期保障）

### 变量池

| 变量 | 内容 |
|------|------|
| **目标受众** | 贸易公司老板、采购总监（需要稳定供应商） |
| **核心痛点** | ① 交期赶不上，客户流失 ② 质量不稳定，反复返工 |
| **破局道具** | ① 交货准时的大卡车 ② 质量检测报告（全部通过） |

### 剧本设计

| 阶段 | 内容 |
|------|------|
| **黄金开场** | 反差特写：合同上红色的"延期罚款"条款被狠狠指着 |
| **核心道具** | 准时交货的照片 or 签收单特写 |
| **情绪弧线** | 紧张（交期压）→ 恐惧（怕延期）→ 转机（供应商承诺）→ 安心（准时交付） |

### 品牌金句

> "说到做到，交期有保障。"

### B-Roll建议

| 阶段 | 建议B-Roll |
|------|-----------|
| 压抑期 | 合同特写、红色罚款条款、日历翻页 |
| 转机期 | 机器全速运转、工人有条不紊 |
| 释放期 | 货车装货出发、签收单特写、质量报告盖章 |
