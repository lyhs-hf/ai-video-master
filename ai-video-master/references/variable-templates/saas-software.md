# SaaS/企业软件模板

> 版本：1.0 | 日期：2026-04-07

---

## 模板A：数据型（CRM/销售管理）

### 变量池

| 变量 | 内容 |
|------|------|
| **目标受众** | 销售总监、销售老板（20-50人团队） |
| **核心痛点** | ① 销售撞单、客户流失无追踪 ② 老板看不到数据，全靠人工汇报 |
| **破局道具** | ① 手机端实时数据看板（绿色上升曲线） ② 客户跟进自动提醒 |

### 剧本设计

| 阶段 | 内容 |
|------|------|
| **黄金开场** | 反差特写：销售盯着电脑，客户被标记"已流失" |
| **核心道具** | 手机端数据看板（绿色曲线飙升） |
| **情绪弧线** | 焦虑（看不到数据）→ 绝望（客户流失）→ 希望（系统预警）→ 释然（曲线上升） |

### 品牌金句

> "您安心盯战略，客户跟进交给我们。"

### B-Roll建议

| 阶段 | 建议B-Roll |
|------|-----------|
| 压抑期 | 电脑屏幕特写、红色流失标记 |
| 转机期 | 手机端数据看板、推送通知 |
| 释放期 | 绿色增长曲线、团队开会看数据 |

## 模板B：效率型（OA/人事/进销存）

### 变量池

| 变量 | 内容 |
|------|------|
| **目标受众** | 企业老板、行政负责人（50-200人公司） |
| **核心痛点** | ① 审批流程慢，员工频繁催 ② 库存混乱，经常断货/积压 |
| **破局道具** | ① 手机审批界面（一键通过） ② 库存预警大屏 |

### 剧本设计

| 阶段 | 内容 |
|------|------|
| **黄金开场** | 灯光巨变：原本堆满文件的桌子，瞬间变成手机屏幕 |
| **核心道具** | 手机审批界面（手指点"通过"） |
| **情绪弧线** | 烦躁（文件堆积）→ 崩溃（不断有人催）→ 转机（一键搞定）→ 清爽（桌面干干净净） |

### 品牌金句

> "您专注决策，流程运转交给我们。"

### B-Roll建议

| 阶段 | 建议B-Roll |
|------|-----------|
| 压抑期 | 堆积的文件、等待审批的员工脸 |
| 转机期 | 手机屏幕特写、一键审批动画 |
| 释放期 | 空旷整洁的办公桌、有序的仓库 |
