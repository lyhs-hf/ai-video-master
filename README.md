# 🎬 AI短视频创作大师 v4.0 (AI Short Video Master)

> **别再用 C 端的情感套路去骗 B 端老板的钱了！也别再让 AI 视频工具瞎跑出肢体畸形的废片了！**

这是一个具有**状态机控制流（State Machine）**和**RAG检索增强**能力的工业级 Agent 技能包（Prompt Framework）。它基于数百次短视频跑图的血泪实测数据打造，将“好莱坞影视编剧法则”与“大模型防崩坏机制”深度缝合。

## 🌟 核心破局能力 (Why this?)

目前市面上的 AI 视频教程都在教你“写唯美的提示词”，但这在 B2B 商业变现中毫无意义。本系统解决了 AI 短视频落地的两大死穴：

1. **解决“商业转化差”**：内置**B2B痛点斩杀模型**，老板只关心“省多少钱、赚多少钱、省多少心”。我们用“红头文件/计算器”等具象化道具代替无效的“微笑和热茶”。
2. **解决“AI 抽卡废片率高”**：建立**工业级防崩坏机制**（场景≤3个、固定运镜优先、动作微小化、Step 0 资产固化防漂移）。

## ⚙️ 系统级架构与特性 (Features)

本技能包基于 **XML标签化架构** 构建，包含四大核心引擎：

* 🧠 **`<Workflow_Engine>` 三阶段状态机工作流**：支持 `[停机等待]` (Human-in-the-loop)，打破大模型一次性输出导致失控的通病。
* 🔍 **`<Role_and_Identity>` 知识库检索驱动 (RAG)**：Agent 强制在创作前向用户收集变量，并检索内置的行业模板库。
* 🛡️ **`<Constraint_Rules>` 钢铁红线自检 (CoT)**：输出前强制进行 Chain of Thought 自检，台词＞15字、场景＞3个直接打回重写。
* 🚑 **`<Troubleshooting_Protocol>` 触发式急救协议 v2.0**：全网首创！当用户遇到肢体融图、文字乱码、角色漂移时，通过自然语言即可触发“微创手术”级的提示词急救方案。

## 🚀 快速部署指南 (Quick Start)

### 方式一：部署为专属 Agent（推荐，零故障体验）
1. 在 **Coze (扣子) / Dify** 中创建一个新的 Bot。
2. 将根目录下的 `SKILL.md` 完整复制，粘贴到 Bot 的 **“人设与回复逻辑 (System Prompt)”** 中。
3. 将 `references`、`industry-cases`、`model-adapters`、`variable-templates` 文件夹下的所有 `.md` 文件上传至该 Bot 的 **“知识库 (Knowledge Base)”** 中。
4. 保存发布，即可拥有一个带 RAG 检索能力的顶级 AI 编剧运营官！

### 方式二：单次对话流使用
直接将 `SKILL.md` 复制发给 Claude / GPT-4 等大模型即可开始工作。
